# Audio Files Setup

To enable sound effects and music, please add the following files to this `public` folder:

1. `music.mp3` - Background music (soft, romantic).
2. `pop.mp3` - Success sound effect (confetti pop).
3. `swoosh.mp3` - Swoosh sound effect (for the "No" button movement).

You can find free sound effects at:
- https://pixabay.com/sound-effects/
- https://freesound.org/
